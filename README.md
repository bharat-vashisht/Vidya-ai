# Vidya AI 🎓
### Empowering Every Student — Anytime, Anywhere!

## Problem Statement
600 million Indians have limited or no internet access. Rural students 
lack access to quality education and personalized learning assistance. 
Vidya AI bridges this gap using the power of Gemma 4.

## Solution
Vidya AI is an AI powered learning assistant built with Gemma 4 that 
provides quality education to every student regardless of their 
location or economic background.

## Features
- 💬 **Ask Vidya** — 24/7 AI teacher in English and Hindi
- 📚 **NCERT Helper** — Class 6-12 curriculum aligned assistance
- 📝 **Exam Preparation** — Auto generated practice questions
- 👨‍🏫 **Teacher Assistant** — Lesson plans and worksheets
- 📊 **Performance Tracker** — Personalized study advice

## Tech Stack
- **Gemma 4** — AI backbone (gemma-4-26b-a4b-it)
- **Python** — Backend
- **Streamlit** — Frontend
- **Google AI Studio** — API access

## Impact
- Helps students from Class 6-12
- Works in English and Hindi
- NCERT curriculum aligned
- Designed for low resource environments
